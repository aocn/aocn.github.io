搭建hexo博客

Hexo搭建步骤

  • 安装Git

    • win：安装包，多使用git bash
    • linux: sudo apt-get install git
    • 测试：git version
      -例如： git version 2.27.0.windows.1

  • 安装Node.js

    • linux
      • sudo apt-get install nodejs
      • sudo apt-get install npm
    • win: 安装包
    • 检测：
      • node -v - v14.16.0
      • npm -v - 6.14.11
    • 关于国内npm被和谐
      • 配置npm的仓库为国内镜像
      • npm config set registry https://registry.npm.taobao.org

  • 安装Hexo并体验Hexo

    • 站点： https://hexo.io/zh-cn/
    • 检测： hexo -v

  • 关键目录（了解）

    • public：存放生成的页面
    • scaffolds：生成文章的一些模板
    • source：用来存放你的文章
    • themes：主题

-GitHub创建个人仓库
- 注册GitHub账户
- New repository，新建仓库
- 创建aocn.github.io，其中aocn位置就是你GitHub的用户名

  • 将SSH添加到GitHub
    • cd ~/.ssh 找到ssh公钥 pub

  • 如果没有生成

  • 将hexo部署到GitHub

    • 打开站点配置文件 _config.yml
    • 按安装部署命令，一键部署
      • npm install hexo-deployer-git –save
  • 其他操作
  • 后续操作，后续再说…
  • https://aocn.github.io
